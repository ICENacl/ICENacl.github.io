Key Highlights

New Models

  • Moss-VL: A new multimodal vision-language model is now supported with
    flashinfer cross-attention custom masks.
  • LTX2.3 HQ Pipeline: High-quality diffusion pipeline support added for LTX
    video generation models.
  • MUSA Backend: Extends SGLang compatibility to Moore Threads MUSA GPUs for
    DeepSeek V2/V3/R1 inference.

Performance Optimizations

  • Breakable Piecewise CUDA Graph: A major experimental feature that allows
    more granular CUDA graph capture, reducing CPU overhead in dynamic-shape
    scenarios.
  • MoE Expert Skipping & Dispatch Refactoring: Reduces wasted computation in
    MoE layers by skipping unselected experts and reorganizing mask handling.
  • XPU Pipeline Parallelism: Intel GPU users can now leverage pipeline
    parallelism for larger model deployments.

Bug Fixes

  • AMD speculative decoding stability improved when using non-default page sizes.
  • MoE no_combine correctness fixed.
  • FP8 checkpoint detection now works with remote HuggingFace Hub IDs.
  • Ascend NPU memory pre-allocation and flash-attention sequence-parallelism
    crashes resolved.
