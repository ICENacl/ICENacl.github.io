SGLang Main Branch Commit Summary (2026-04-30 UTC+8)

Overview

This report summarizes 38 commits merged into the SGLang main branch during UTC+8 2026-04-30 00:00–24:00. The commits span bug fixes, model support expansion, performance optimizations, CI improvements, and infrastructure refactoring across GPU (NVIDIA), AMD, NPU, XPU, and Apple Silicon backends.

Highlighted Changes

New Model Support

  • Qwen3ForPooledOutput now forwards get_input_embeddings to its inner model.
  • LoRA support added for Qwen3.5 and Nemotron3, including new model files and comprehensive logprob-diff tests.
  • MiniMax-M2 (NPU) received optimized fused ops (split_qkv_tp_rmsnorm_rope).
  • MOSS-VL fixed to use Conv3dLayer and removed a no-op flat_encoder_result path.

Performance Optimizations

  • Apple Silicon / MLX: Introduced async-eval-based decode partial-overlap scheduling, significantly improving generation throughput on MLX backends.
  • ModelExpress: Added peer-to-peer RDMA transfer support with configurable transport backends (transfer_engine or nixl).
  • AWQ Quantization: Major refactor splitting kernel calls, weight initialization, and schemes (linear, Marlin, MoE) into modular files.
  • AMD MoriEP: Enabled SDMA path for MoE token dispatch.
  • XPU: Added deterministic-operation mode for reproducible results.

Bug Fixes

  • Speculative Decoding: SGLANG_ENABLE_SPEC_V2 is now enabled by default; fallback-to-v1 logic cleaned up and overlap-scheduler compatibility improved.
  • MoE & CUDA Graphs: Fixed compatibility between --moe-dense-tp-size 1 and piecewise CUDA graph capture.
  • LoRA & CUDA Graphs: Eliminated a CUDA-graph-breaking scalar assignment in seg_indptr.
  • Disaggregation / PD: Fixed mamba cache capping in decode mode and added missing index/KV transfer for MTP layers in NSA disaggregation.
  • Memory Cache: Corrected host-leaf status check from “evicted” to “backuped” in the hierarchical radix cache.
  • FP8 / Qwen3.5: Fixed per-tensor scale broadcasting in _make_packed_weight_loader.
  • Context-Length Handling: Requests whose total token count exactly equals context_len are now accepted.
