八、总结

在 2026-05-05 这一天，SGLang main 分支的活跃方向主要集中在：

  1. 扩散模型（Diffusion）生态持续完善：HunyuanVideo FP8 量化、LTX2 rotary fusion、encoder 缓存、USP 通信优化等改动显著提升了扩散推理的吞吐量与易用性。
  2. 多硬件平台扩展：Intel XPU 新增 DeepSeek V3.2 支持；AMD 在 Qwen3.5 FP8/MXFP4 上启用 EAGLE 推测解码，并修复了若干平台专属 bug。
  3. 核心引擎优化：MoE LoRA 同步屏障移除、FlashInfer autotune 缓存、specdec CPU 开销降低等改动直接改善了 LLM serving 的延迟与吞吐。
  4. 可观测性增强：SWA / Mamba 缓存指标暴露、HiCache cached_tokens_details 回流，使线上运维与调试更加透明。
  5. 基础设施与文档：K8s 集成测试、B300 Pro 精度验证配置、Gemma4 MTP cookbook 等降低了生产环境部署门槛。
