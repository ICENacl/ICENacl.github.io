总结

昨日 47 个 commit 的主要亮点：

  1. HiSparse 支持 FP8 KV cache：通过路由到 flashmla_kv 后端，HiSparse 现在可以配合 FP8 KV cache 使用，扩展了性能和内存优化空间。
  2. Indexer-topk 捕获基础设施：新增 --enable-return-indexer-topk 参数，为 V3.2 NSA 模型提供 indexer topk 索引返回能力，并统一了 state capturer 目录结构。
  3. SWA HiCache 支持：统一 radix cache 新增 SWA HiCache 支持，增强了滑动窗口场景的缓存能力。
  4. Dumper Grafter 大规模扩展：连续 7 个 commit 大幅增强 dumper grafter 调试工具链的功能。
  5. Weight Checker 大量增强：6 个 commit 完善 weight checker 的 buffer pattern、checksum、FP8 等处理能力。
  6. PD 分离稳定性修复：多个 commit 修复 PD disaggregation 中的状态更新、指标统计和异常处理问题。
